# Netflix Clone

Netflix clone built for practice

## Libraries/Tools ##
* ReactJS
* HTML5
* CSS3
* Firebase
* Yarn
* Babel

## Run Application ##
1. Yarn installations of dependencies
      ``` 
      yarn install 
      ```
3. Build the application
      ```
      yarn run build 
      ```
4. Start the application
      ``` 
      yarn run start 
      ```
7. Navigate to ```http://localhost:3000```

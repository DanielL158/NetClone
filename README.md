# Netflix Clone

Netflix clone built for practice

## Libraries/Tools ##
* ReactJS
* HTML5
* CSS3
* Firebase
* Npm
* Babel

## Run Application ##
1. NPM installations of dependencies
      ``` 
      npm install 
      ```
3. Build the application
      ```
     npm run build 
      ```
4. Start the application
      ``` 
      npm run start 
      ```
7. Navigate to ```http://localhost:3000```
